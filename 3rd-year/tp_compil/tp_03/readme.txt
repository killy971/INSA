Pour tester la classe, veuillez utiliser le fichier batch dans le dossier files/
(avec une fenêtre DOS en s'étant préalablement positionné dans le repertoire files/)

Utilisation de la classe Analyse:
java Analyse fichierInput.xml fichierOutput.html

Fonctionnalitées supplémentaires:
- gestion simplifiée du wrap des lignes de code source html généré
(taille de ligne jamais supérieure à environ 60 caractères)
- calcul du temps de traitement

Fichiers d'exemples fournis:
- exemple.xml : exemple simple.
- exemple_long.xml : exemple pour tester la robustesse et la vitesse
du programme sur un document long.
- exemple_profond.xml : exemple pour tester la robustesse et la vitesse
sur un document ayant une grande profondeur d'imbrication de paragraphes.
